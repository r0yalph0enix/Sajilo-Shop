# Sajilo Shop

"Sajilo Shop" is a web based application which helps in purchasing electronic items  via  online medium.<br>
It has paypal payment system integrated on it.<br>
This system  can recommend similar items based on category and brand with the help of cosine similarity.<br>
This system is developed using HTML,CSS,Javascript,PHP & MySQL.<br>
It includes mainly two sides admin & user.

<h4>Admin</h4><br>
It holds the key  for editing and updating info.<br>
It manages the whole system data like user data,billing data and price detail info of electronic items.

<h4>User</h4><br>
It can register,login and update its profile.
It can view the list of  item,search it  and purchase it.
